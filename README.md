# Práctica IAM (Identity and Access Management)

Este repositorio contiene la evidencia de la práctica de IAM realizada en Google Cloud Platform (GCP).

## Contenido

* `gcloud-iam.md`: resultado de la configuración de una cuenta de servicio con permisos de solo lectura y evidencia de la política IAM obtenida mediante Google Cloud CLI.

## Objetivo

Aplicar el Principio de Menor Privilegio mediante la creación de una cuenta de servicio y la asignación del rol Viewer para realizar tareas de auditoría sin otorgar privilegios administrativos innecesarios.

## Herramientas utilizadas

* Google Cloud Platform (GCP)
* Google Cloud CLI (gcloud)
* Ubuntu Linux
* GitHub
